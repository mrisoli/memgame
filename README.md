# Memgame

Simple memory game written in React and NextJS

## Requirements

node 15.7.0 or Docker

## How to use

Running locally:

```
$ yarn
$ yarn dev
```

Running with docker

```
$ docker build -t app .
$ docker run --rm -it \
  -p 3000:3000
  app
```

A demo version is deployed at:

https://memgame.vercel.app
